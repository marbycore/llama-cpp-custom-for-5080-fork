# Release v1.2.0: Blackwell Discovery & Portability

## ✨ Highlights
- **Universal LAN Support**: Automatic IP discovery for local networks.
- **iPhone / Off Grid Integration**: Identity Shim for seamless mobile discovery.
- **RTX 5080 Blackwell Optimized**: Custom build with Flash Attention 3 and FP4 support.
- **Zero-Friction Launch**: Refined desktop shortcuts with a new OLED Dashboard.

## 📦 What's inside the ZIP?
- `llama-server.exe`: High-performance binary.
- `Llama-Server_RTX5080.bat`: Main launcher.
- `ollama_shim.js`: Protocol translator for mobile apps.
- `sync_hermes.ps1`: Synchronization bridge for Hermes TUI.

## 🚀 Quick Start
Download and extract the ZIP, then run `Llama-Server_RTX5080.bat` to begin.
